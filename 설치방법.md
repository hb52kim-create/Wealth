# WEALTH — 내 자산 포트폴리오 PWA

## 📱 아이폰 앱처럼 설치하는 법 (GitHub Pages)

### 1단계 — GitHub 계정 만들기
https://github.com 에서 무료 계정 가입

### 2단계 — 새 저장소(Repository) 만들기
- 우측 상단 + 버튼 → "New repository"
- Repository name: `wealth-portfolio` (아무 이름도 OK)
- Public 선택
- "Create repository" 클릭

### 3단계 — 파일 올리기
- "uploading an existing file" 링크 클릭
- 이 폴더 안의 파일 4개를 모두 드래그앤드롭:
  - index.html
  - manifest.json
  - sw.js
  - icon-192.png
  - icon-512.png
- "Commit changes" 클릭

### 4단계 — GitHub Pages 활성화
- 저장소 → Settings 탭
- 왼쪽 메뉴 "Pages" 클릭
- Source: "Deploy from a branch"
- Branch: main / (root)
- Save 클릭
- 1~2분 후 URL 생성됨: https://[내아이디].github.io/wealth-portfolio

### 5단계 — 아이폰에 설치
1. 아이폰 사파리에서 위 URL 접속
2. 하단 공유 버튼(□↑) 탭
3. "홈 화면에 추가" 선택
4. "추가" 탭
5. 홈화면에 WEALTH 아이콘 생성 완료! 🎉

## 💡 참고
- 데이터는 기기 로컬(localStorage)에 저장됨 → 완전히 내 것
- 오프라인에서도 마지막 데이터로 앱 실행 가능
- 가격은 인터넷 연결 시 자동 갱신 (1분 주기)
- 여러 기기에서 쓰려면 각 기기에서 설치하고 자산 입력 필요
